In Design, function rules

I experience Github as a user, but I analyze it as a designer. I see how every part of it has been designed to ease the friction between a highly abstract concept such as version control, and the everyday use that not only programmers make of Github. From forking to publishing, Github is privileging function, making user-centric design a standard.
